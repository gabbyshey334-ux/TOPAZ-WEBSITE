# TOPAZ 2.0 Website

This repo contains the TOPAZ 2.0 dance competition website.

## Vercel deployment

The app lives in the **`app/`** folder. In Vercel:

1. **Project Settings → General → Root Directory** → set to **`app`** and Save.
2. Redeploy (Deployments → ⋮ on latest → Redeploy).

Build command: `npm run build`  
Output directory: `dist`
