# Event images

Event cards and the homepage event section use images from this folder.

## Expected files

- **trophy-gold.jpg** — Main trophy/competition image (e.g. gold eagle trophy). Used for “The Return of TOPAZ 2.0” and as fallback.
- **competition-2026.jpg** — General competition or stage photo for other events.
- **stage-performance.jpg** — Stage/performance photo (e.g. for Summer Intensive).

## Specs

- Format: JPG or PNG  
- Size: 1200×800px minimum  
- Aspect ratio: 3:2 or 4:3  
- File size: &lt; 500KB (optimized)

If a file is missing, the site falls back to a placeholder image. Add these files to get your own event imagery on the Schedule and Home pages.
